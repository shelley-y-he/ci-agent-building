- **Tier** indicates extraction depth: `Deep` / `Standard` / `Skim`
- **Tags** are method families and topics (e.g., `DiD`, `synthetic-control`, `staggered`, `sensitivity`)
- To add a paper, run extraction via subagent and append a row here
